# Maven Wrapper Commands

| Command                      | Description                   |
| ---------------------------- | ----------------------------- |
| `./mvnw`                     | Run default goal (compile)    |
| `./mvnw clean`               | Clean target directory        |
| `./mvnw compile`             | Compile source code           |
| `./mvnw test`                | Run tests                     |
| `./mvnw package`             | Build JAR file                |
| `./mvnw install`             | Install package to local repo |
| `./mvnw dependency:resolve`  | Download dependencies         |
| `./mvnw dependency:tree`     | Show dependency tree          |
| `./mvnw spring-boot:run`     | Run Spring Boot app           |
| `./mvnw -DskipTests package` | Build without running tests   |
| `./mvnw dependency:analyze`  | Analyze unused dependencies   |
