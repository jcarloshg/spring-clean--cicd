### Hibernate Best Practices

1. Use Session management properly:
   - Prefer `getCurrentSession()` for transaction-scoped sessions
   - Always close sessions properly (try-with-resources or finally blocks)

2. Use LAZY loading wisely:
   - Use `@OneToMany(fetch = FetchType.LAZY)` to avoid loading entire object graphs
   - Use `@Fetch(FetchMode.SUBSELECT)` to avoid N+1 queries
   - Initialize lazy collections with `Hibernate.initialize()` or fetch joins

3. Avoid N+1 query problems:
   - Use `JOIN FETCH` in JPQL/HQL for eager loading related entities
   - Use `@BatchSize(size = N)` for batch loading lazy collections
   - Monitor SQL logs during development

4. Use proper transaction handling:
   - Keep transactions short (avoid long-running operations)
   - Use appropriate isolation levels
   - Never commit/rollback in data access layer; delegate to service layer

5. PreferCriteria API or JPQL over native SQL when possible:
   - Better portability across databases
   - Type safety with Criteria API
   - Named queries for reusable statements

6. Handle exceptions properly:
   - Do NOT catch Hibernate exceptions and continue using the session
   - `HibernateException` marks the session as invalid
   - Use `@Repository` with proper exception translation via `@RepositoryAttributes`

7. Use proper caching strategies:
   - Enable L2 cache for frequently accessed, rarely modified data
   - Use `@Cache(usage = CacheConcurrencyStrategy.READ_WRITE)`
   - Monitor cache hit/miss ratios

8. Optimize batch operations:
   - Use `session.setBatchSize(N)` for batch inserts/updates
   - Use `Hibernate.jdbc.batch_size` configuration
   - Clear/persist in batches to avoid memory issues

9. Use proper fetch strategies:
   - Use `FetchType.LAZY` by default
   - Only use `EAGER` for truly required associations
   - Use named entity graphs for specific fetch requirements

10. Always use meaningful indexes:
    - Add indexes on foreign key columns
    - Use `@Index` annotation for composite indexes
    - Analyze query execution plans for large tables
